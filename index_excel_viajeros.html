⚙️ Configurar Firebase
🔥
Necesitás configurar Firebase para usar la app completa.
  1. Ingresá a console.firebase.google.com
  2. Creá un proyecto nuevo (ej: "ArmoTuViajeMagico")
  3. Andá a Configuración del proyecto → General → Tus apps → Web
  4. Registrá la app y copiá el firebaseConfig
  5. En Firestore Database, creá la base de datos en modo de prueba
  6. Pegá los datos acá abajo y reemplazá en el código HTML

Los datos quedarán guardados en tu navegador y el código HTML mostrará cómo reemplazarlos de forma permanente.
